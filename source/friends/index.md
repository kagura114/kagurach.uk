## 添加指南
1. fork 本仓库 -> [这里](https://github.com/kagura114/kagurach.uk)
2. 修改 `source/_data/friends.yml` [文件](https://github.com/kagura114/kagurach.uk/blob/main/source/_data/friends.yml) 
  ```yaml
  title: 你的名字
  avatar: 你的头像URL
  url: 你的主页URL
  description: 你的描述
  ```
3. 预览效果
  在仓库根目录下执行下面的命令预览
  ```sh
  ./build.sh -d
  ```
4. 提交 PR 等待我合并
